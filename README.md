## Getting Started

First, run the development server:

```bash
npm run install
# then
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Features

- Todo List
- Add Todo
- Delete Todo
- Store todo data on Easy Peasy
- Persist data using Easy Peasy
